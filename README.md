# AssetsHub

AssetsHub adalah aplikasi berbasis Flutter yang digunakan untuk mengelola aset dan inventaris secara efisien dengan fitur pencatatan barang, peminjaman, serta pelacakan status barang secara real-time.

## 🚀 Teknologi yang Digunakan
- **Flutter** dengan **GetX** untuk state management
- **Firebase** untuk real-time database dan autentikasi
- **Flutter Animate** untuk animasi UI yang halus

## 🔧 Cara Instalasi
1. Clone repository ini:
   ```sh
   git clone https://github.com/Sidqii/InventoryApp.git
   ```
2. Masuk ke direktori proyek:
   ```sh
   cd pusdatin_end
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Jalankan aplikasi:
   ```sh
   flutter run
   ```
